# 📄 Desafio 001 - Módulo 1 - Capítulo 5

## 📌 Enunciado

Desafio proposto no material **"Desafio das Mensagens"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 Material original:
https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-01/d001/desafio-mensagens.pdf

## 💡 Objetivo

Criar uma página HTML utilizando apenas parágrafos e quebras de linha para exibir mensagens organizadas.

## 🌐 Visualização

👉 https://github.com/Giulia-MB/FrontEnd-NTTdata/blob/main/html-semantico/desafios/d001/index.html

## 🛠️ O que foi praticado

* Uso da tag `<p>` para separar blocos de conteúdo
* Uso da tag `<br>` para quebra de linha
* Organização de conteúdo sem uso excessivo de quebras
* Estruturação visual simples com HTML

## 📚 Aprendizados

* Diferença entre parágrafos e quebras de linha
* Importância de evitar múltiplos `<br>` para espaçamento
* Organização semântica do conteúdo
* Como estruturar texto de forma clara no HTML

## 🚀 Resultado

Página HTML exibindo mensagens organizadas conforme o desafio proposto.
