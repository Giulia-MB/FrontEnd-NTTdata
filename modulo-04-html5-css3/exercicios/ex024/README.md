## 📚 Aprendizados

* `iframe` permite exibir outra página dentro da página atual  
* `src` define o endereço carregado no frame  
* `width` e `height` controlam dimensões  
* `name` permite abrir links dentro do iframe  
* `target="frame"` envia o link para o quadro desejado  
* `srcdoc` permite escrever HTML direto dentro do iframe  
* `sandbox` restringe ações do conteúdo incorporado  
* `referrerpolicy` controla informações enviadas ao site carregado  
* `95vw` significa 95% da largura visível da tela  
* Iframes também podem incorporar:
  - vídeos
  - mapas
  - documentos
  - apresentações online
