## 🛠️ O que foi praticado

* Estrutura básica com `<form>`
* Uso de `action`, `method="get"` e `method="post"`
* Campos com `input type="text"`
* Campos de senha com `password`
* Campos numéricos com `number`
* Datas com `date`, `month` e `time`
* Campos de e-mail e telefone
* Validações com `required`, `min`, `max`, `maxlength`, `minlength`
* Placeholder e autocomplete
* Botões `submit` e `reset`
* Agrupamento com `fieldset` e `legend`
* Escolha única com `radio`
* Escolhas múltiplas com `checkbox`
* Upload de arquivos com `file`
* Seletor de cor com `color`
* Barra deslizante com `range`
* Listas com `select`, `option` e `optgroup`
* Sugestões com `datalist`
* Área de texto com `textarea`
* Resultados dinâmicos com `output`
* Interação com JavaScript via `oninput`

## 📚 Aprendizados

* `form` cria a estrutura do formulário  
* `action` define para onde os dados serão enviados  
* `method="get"` envia dados pela URL  
* `method="post"` envia dados de forma mais adequada para dados sensíveis  
* `label` melhora acessibilidade e usabilidade  
* `name` identifica o campo no envio  
* `id` conecta o campo ao label  
* `required` torna o preenchimento obrigatório  
* `placeholder` exibe dica no campo  
* `autocomplete` ajuda no preenchimento automático  
* `fieldset` organiza grupos de informações  
* `output` mostra resultados em tempo real  
* JavaScript pode interagir diretamente com campos do formulário
