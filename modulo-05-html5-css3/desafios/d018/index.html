Projeto Cordel Moderno

Projeto construído para praticar a criação de conteúdo em formato de cordel utilizando efeito Parallax.

Saiba mais...

Esse é um projeto feito para transformar o Cordel Moderno, de Milton Duarte, em uma página web com visual mais elaborado, misturando poesia, imagens e efeitos de rolagem. A página é dividida em seções, alternando partes com fundo simples e partes com imagens de fundo. A ideia foi criar uma leitura mais visual, usando o efeito parallax para dar profundidade enquanto o usuário rola a página.

projeto Redes Sociais

Esse é um projeto de feito para simular uma tela de celular com atalhos para redes sociais

Saiba mais...

A ideia foi criar uma interface visual e interativa usando HTML, CSS, imagens e iframe. A página principal mostra um celular no centro da tela e alguns botões laterais com ícones de redes sociais. Ao clicar em cada ícone, a tela dentro do celular muda, exibindo uma prévia da rede escolhida. Em cada tela, o usuário deve rolar com o mouse ou com o touchscreen para chegar até o botão "ACESSE", que leva para o link externo correspondente.

projeto Login Responsivo

Esse é um projeto feito para criar uma tela de login responsiva, adaptada para diferentes tamanhos de tela.

Saiba mais...

A ideia foi praticar Mobile First, Media Queries e estilização de formulários com HTML e CSS. A página apresenta um card de login centralizado, com imagem decorativa, campos de e-mail e senha, botão de entrada e link para recuperação de senha. O layout muda conforme o tamanho da tela, ficando mais compacto no celular e mais amplo em tablets e desktops.

projeto Álbum de Fotos

Esse é um projeto feito para criar um álbum de fotos responsivo usando HTML e CSS.

Saiba mais...

A ideia foi organizar várias imagens em colunas que se adaptam ao tamanho da tela. Em telas menores, as imagens aparecem em uma única coluna; em tablets, passam a ocupar duas colunas; e em telas maiores, o layout se organiza em quatro colunas.
