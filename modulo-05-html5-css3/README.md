## 📚 Aprendizados

* O **Flexbox** trabalha com dois eixos:
  * eixo principal, chamado de **main axis**
  * eixo cruzado, chamado de **cross axis**

* `flex-direction` define a direção do eixo principal:
  * `row`
  * `column`

* `flex-wrap` permite que os itens quebrem linha quando necessário  

* `flex-flow` combina:
  * `flex-direction`
  * `flex-wrap`

* `justify-content` alinha os itens no eixo principal  

* `align-items` alinha os itens no eixo cruzado  

* `align-content` controla o espaço entre múltiplas linhas  

* `align-self` permite alinhar um item específico  

* `flex-basis` define o tamanho inicial do item  

* `flex-grow` controla quanto o item pode crescer  

* `flex-shrink` controla quanto o item pode encolher  

* `flex` é o shorthand para:

```css
flex: grow shrink basis;
```
