# 📄 Desafio 011 - Módulo 03 - Capítulo 19

## 📌 Enunciado

Desafio proposto no material **"Desafio do Astronauta"** do curso de HTML e CSS da plataforma **Curso em Vídeo**.

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-02/d011/desafio-astronauta-1.pdf)

## 💡 Objetivo

Unir duas imagens fornecidas no desafio — uma imagem de fundo espacial e um astronauta com transparência — criando uma página onde o fundo ocupe toda a tela e o astronauta permaneça centralizado, mesmo ao redimensionar a janela.

## 🌐 Visualização

👉 [Abrir desafio](https://giulia-mb.github.io/html-semantico/modulo-03-html5-css3/desafios/d011/index.html)

## 🛠️ O que foi praticado

* Uso de imagem de fundo em tela cheia  
* Centralização de elementos com CSS  
* Sobreposição de imagens  
* Responsividade com adaptação ao tamanho da tela  
* Posicionamento absoluto e relativo  
* Controle de fundo com cover e centralização  

## 📚 Aprendizados

* Como aplicar imagens de fundo ocupando toda a viewport  
* Diferença entre posicionamento relativo e absoluto  
* Técnicas para centralizar elementos na tela  
* Importância da transparência em imagens PNG  
* Adaptação visual para diferentes resoluções  
* Organização de layout simples com foco visual  

## 📄 Estrutura do desafio

### `index.html`

Página contendo:

* fundo espacial em tela cheia  
* astronauta centralizado  
* composição visual com duas imagens  

### Arquivos utilizados

* `espaco.jpg` → imagem de fundo  
* `astronauta.png` → imagem principal com transparência  

## 🚀 Resultado

Página visual temática com efeito espacial, combinando imagens e posicionamento centralizado de forma responsiva.

## 💬 Observação

Este desafio é excelente para praticar posicionamento em CSS e composição visual com imagens, conceitos muito utilizados em banners, capas de sites e interfaces modernas.
